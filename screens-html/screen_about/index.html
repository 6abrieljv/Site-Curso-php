Sobre o NerdHub

Nossa Missão

Nossa missão é criar uma plataforma centralizada e colaborativa para os estudantes de Ciência da Computação da Estácio São Luís. O NerdHub nasceu da necessidade de um espaço unificado para compartilhar conhecimento, organizar materiais de estudo, divulgar eventos e fortalecer a comunidade acadêmica do nosso curso.

O Projeto

Este site é um projeto acadêmico desenvolvido como parte da curricular do curso. Ele representa a aplicação prática de conceitos de desenvolvimento web, design de interfaces e gestão de projetos, utilizando tecnologias modernas como HTML, CSS com TailwindCSS e JavaScript.
